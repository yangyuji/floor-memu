# floor-memu
电商网站里面的H5活动页面常用的楼层定位菜单实现

## 实现功能
> *  移动端页面菜单，滚动的时候固定在页面顶部，根据滚动位置动态改变菜单样式

## 实现技术
> *  原生实现，无依赖；

## 特点
> *  代码非常小，只有100多行，每个人都能看懂，可以进行扩展和修改；
> *  无依赖，这样你就可以用于任何地方了；
